# Rule: One Repo, One Task

Claude Code operates on SmartFlowSite only when invoked in this repo.

Do not run commands, edit files, or make changes in any other repo.

One approved task at a time. Do not infer or self-assign follow-on tasks from the current task.

Do not run blanket commands across multiple repos or files (e.g. `git add .`, `npm install` across all repos).
