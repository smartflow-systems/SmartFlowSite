## 🔗 INTEGRATION ARCHITECTURE

### **How Everything Connects:**

```
                  [MASTER BRAIN - SmartFlowSite]
                              │
        ┌─────────────────────┼─────────────────────┐
        │                     │                     │
    [AI Layer]          [API Gateway]        [Services Layer]
        │                     │                     │
  ┌─────┴─────┐         ┌────┴────┐         ┌──────┴──────┐
  │           │         │         │         │             │
Claude    ChatGPT    REST API   WebHooks   Shared DB   Cache
  │           │         │         │         │             │
  └─────┬─────┘         └────┬────┘         └──────┬──────┘
        │                    │                     │
    [Orchestrator]      [Connectors]        [State Store]
        │                    │                     │
        └────────────────────┼─────────────────────┘
                             │
        ┌────────────────────┼────────────────────┐
        │                    │                    │
   [Products]          [Infrastructure]     [Marketing]
        │                    │                    │
   ┌────┴────┐          ┌────┴────┐         ┌────┴────┐
   │         │          │         │         │         │
 CRM    Analytics   Design Sys  Deploy   Growth    Video
```

### **Data Flow Patterns:**

1. **Unified Authentication:**
   - JWT tokens shared across all repos
   - Single sign-on (SSO) capability
   - Centralized user management from SmartFlowSite

2. **Shared Theme System:**
   - `sfs-theme-package` → CSS variables + components
   - `sfs-brand-assets` → Logo, colors, typography
   - `sfs-design-system` → React/Vue components
   - **Every repo imports the same design tokens**

3. **API-First Integration:**
   - All products expose REST APIs
   - SmartFlowSite acts as API gateway
   - Webhooks for real-time events
   - Microservices architecture

4. **AI Orchestration:**
   - Claude for code generation & analysis
   - ChatGPT for content creation
   - Custom agents for specific tasks
   - Automated workflows via `.sfs/workflows/`
