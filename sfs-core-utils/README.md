# SFS Core Utilities

Shared utilities, scripts, docs, and themes for SmartFlow Systems.

## Structure
- `/scripts` - Deployment, security, git automation
- `/docs` - Runbooks, guides, knowledge base
- `/themes` - Brand CSS/JS files
- `/configs` - Shared configs (.prettierrc, etc.)

## Usage
This repo is NOT deployed. It's a shared library used by other SFS products.
