# SmartFlow Cursor Rules (SFS)
- Enforce SmartFlow palette: black #0D0D0D, brown #3B2F2F, gold #FFD700 (hover #E6C200), beige #F5F5DC, white #FFFFFF
- Prefer glass-card UI, clean spacing, minimal clutter
- Keep routes/docs under [docs/], marketing under [marketing/]
- Never hardcode secrets; use env vars + Replit/GitHub secrets
- Keep changes small, testable, and consistent across repos
